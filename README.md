# 一人称視点の暗号推理ゲーム

## 概要

このプロジェクトは、Next.js / Reactで制作する一人称視点の暗号推理ゲームです。

プレイヤーは椅子に縛られた状態で目を覚まし、謎の男が提示する暗号文と日本語訳の例文を手がかりに、暗号単語の意味を推理して解答します。

## 使用技術

- Next.js
- React
- TypeScript
- CSS Modules

## 必要な環境

- Node.js
- npm
- Git

## セットアップ

初回のみ、以下を実行します。

```bash
npm install
```

## 起動方法

開発サーバを起動します。

``` bash
npm run dev
```

起動後、ブラウザで以下を開きます。

``` bash
http://localhost:3000
```
